PID correction factor calculator (draft)

How this works
  1. All correction factors in the sheet are on the same relative scale (typically vs isobutylene). Ratios between chemicals stay the same no matter which one you treat as “1”. If A, B, C have CFs 1, 2, 4, then relative to B: C is 4÷2 = 2×; relative to A: C is 4÷1 = 4× — exactly as you described.
  2. Pick a reference gas that matches how you read the screen: either Isobutylene scale (CF = 1) for the usual ppm display, or another listed compound if you want the reading to mean “ppm of that reference.”
  3. Equivalent ppm for each row (pure vapor scenario): reading × CFref ÷ CFcompound. When the reference is isobutylene (CFref = 1), that is the same as reading ÷ CFcompound.
  4. Only chemicals with a usable CF for this device + PID sensor are listed. The CF ÷ ref column is CFcompound ÷ CFref (how many times more sensitive than the reference, on this scale).
  5. Real air is often a mixture; each row is still a what-if, not a species ID.
Beta — verify before relying on this. Mixtures, humidity, lamp aging, and manufacturer-specific CF definitions are not modeled. Compare “equivalent ppm” to PEL / REL / TLV / IDLH / PAC for context only.
